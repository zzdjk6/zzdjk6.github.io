Find Android Keystore And Password Back 2014-09-11

I lost an android keystore file but I need it to publish app update to google play store. (I replaced it with new one carelessly.)

I found I’m not the only one have this issue, most of people say you can do nothing, but I found it back successfully.

  1. use Data Rescue to find the key store file back.

    1. generate some keystore files and use “FileIQ” function of this software to train it recognize keystore file type.
    2. change search engine preference to set the limit size of keystore file, I use 100KB, maybe 1000KB is better.
    3. disable other file type recognition to speed up reconstruction phase.
    4. start a deep scan and wait for hours.
    5. you will find some files in “User Defined” nodes, recover them all.

  2. use Android Keystore Password Recovery to check these file one by one, see if it is the one you need. (you don’t have to know the password of these key store, just use a dict.txt and “-m 2” to see the key alias and create date of the keystore file, this tool can show these basic information even the password is wrong.)
    If you’re lucky like me, you find your keystore file back now.

If you forgot your password, you have to go further, continue use Android Keystore Password Recovery or other tools to find your password back.

Good luck!
